# 도둑 포커

### 프로그램 실행
1. 레포지토리를 clone해서 로컬로 갖고 간다
2. pygame 모듈을 설치한다. cmd창이나 powershell에 다음을 입력한다.
```powershell
    pip install pygame
```
3. main.py 프로그램을 실행한다.
아주 쉽다 !

혹시 실행이 안되면 pycache 폴더를 지워보자

------------
현재 껍데기만 만든 상태입니다.

그리고 승패 상관없이 카드 교환이 이루어집니다.
